Foods API Dcumentaion

Foods Routes

Method URL Body Description
GET /foods Get a list of all the foods, will present 10 results per page
GET

/foods/search?s={keyword(Name,Info)}

/foods/search?s=cheese		 search for a food or a food description/info, will present 10 results per page
GET
  1. /foods?page=2
  2. /foods?perPage=5
  3. /foods?reverse=yes
  4. /foods?sort=price
  1. Get a list of all the foods, will present 10 results in page 2
  2. Get a list of all the foods, will present 5 results in page
  3. Get a list of all the foods, will present 10 results in page in reverse order (default sort=price
  4. Get a list of all the foods, will present 10 results in page, Defines what the list will be sorted by
GET

/foods/category_id/{category name}

/foods/category_id/dessert		 Get a list of all the foods in selected category, will present 10 results per page
GET

/foods/prices?min={value}&max={value}

/foods/prices/?min=20&max=35 		Get a list of all the foods priced between 20 and 35, will present 10 results per page(you can add only minimum price or maximum as well)
POST /foods

PUT /foods/id(of food) 
{
"name": "Bulgarian cheese"", (required)
"info": "Info about the food",(required)
"category":  "diary productsd",(required)
"img_url": "image.jpg",
"price": 27 (required)
}
Update a food in the data base, You need a token for this query
DELETE /foods/id(of food) Delete a food from the data base, You need a token for this query
